Внедрение программного обеспечения

Перед автоматизацией необходимо провести анализ оборудования и программного обеспечения в аптечном учреждении на совместимость с программой «F3TAIL». Услуга «Обследование объекта» подразумевает под собой выезд специалиста компании «Спарго Технологии», который даст необходимые рекомендации для подготовки предприятия к автоматизации.

Этапы внедрения:

1. «Обследование объекта»

  • анализ оборудования перед автоматизацией аптечного учреждения;
  • анализ локальной сети перед автоматизацией аптечной сети;
  • анализ ОС.

2. Внедрение программного обеспечения

  • обследование бизнес-процессов заказчика на предмет соответствия функциональным возможностям программного продукта;
  • проведение технического обследование компьютерной техники Заказчика с составлением Протокола;
  • установка на компьютерной технике Заказчика специализированного программного обеспечения;
  • настройка совместимости программного обеспечения с торговым оборудованием;
  • консультации специалистов Заказчика по работе с программным продуктом;
  • консультация Заказчика по выбору торгового оборудования, выбору поставщика программного обеспечения MS Office.

3. Установка и настройка программного обеспечения и оборудования

  • разработка и согласование сметы необходимого оборудования и программного обеспечения;
  • поставка торгового оборудования и компьютерной техники и организация монтажа ЛВС;
  • установка и настройка согласованного с Исполнителем компьютерного и торгового оборудования;
  • установка на рабочую станцию или сервер лицензионной операционной системы, предоставленной Клиентом (из следующего набора: Windows 98, Windows 2000, Windows XP, Windows 2000 Server).

4. Обучение работе с ПО

  • курс обучения работе с системой групп сотрудников Клиента численностью до 5 человек;
  • дополнительное индивидуальное обучение сотрудников аптеки (в составе группы численностью до 3 человек).

5. Техническая поддержка в процессе запуска ПО в эксплуатацию

  • техническая поддержка при проведении инвентаризации;
  • участие в проведении инвентаризации;
  • техническая поддержка в течение первых дней запуска ПО в эксплуатацию (после завершения всех работ по установке и настройке ПО).

6. Настройка ПО под требования клиента

  • настройка ПО в соответствии с индивидуальными требованиями Заказчика без изменения структуры документов, баз данных и базовых алгоритмов.
